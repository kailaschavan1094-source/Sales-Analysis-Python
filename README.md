# 📊 Sales Analysis Using Python

## 📌 Project Overview

This project analyzes sales data using Python. It performs data cleaning, data merging, feature engineering, business KPI analysis, and data visualization to identify important business insights. The project also exports a cleaned dataset for further analysis.

---

## 🎯 Project Objectives

- Analyze sales performance
- Find top-performing states and customers
- Identify profitable and loss-making categories
- Visualize sales trends
- Export a clean dataset for future analysis

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VS Code

---

## 📂 Dataset

The project uses three CSV files:

- List of Orders.csv
- Order Details.csv
- Sales target.csv
---

## 📈 Business KPIs

- Total Sales
- Total Profit
- Total Orders
- Total Quantity Sold
- Average Sales
- Average Profit

---

## 📊 Visualizations

The project includes the following charts:

- Top 10 States by Sales
- Monthly Sales Trend
- Category Wise Sales
- Category Wise Profit
- Top 10 States by Profit
- Top 10 Customers by Sales
- Top 10 Customers by Profit
- Top 10 Sub-Categories by Sales
- Top 10 Sub-Categories by Profit
- Monthly Profit Trend
- Sales vs Profit Scatter Plot
- Correlation Heatmap

## ▶️ How to Run

1. Clone this repository

```bash
git clone https://github.com/your-username/Sales-Analysis-Python.git
```

2. Install required libraries

```bash
pip install -r requirements.txt
```

3. Run the project

```bash
python analysis.py
```

---

## 📊 Key Insights

- Madhya Pradesh generated the highest sales.
- Maharashtra achieved the highest overall profit.
- Electronics was the highest-selling category.
- Some sub-categories generated consistent losses.
- Sales and Profit show a positive correlation.
- Monthly sales and profit trends reveal seasonal business patterns.
